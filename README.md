 Mars Competition – ReqnRoll Selenium Automation (.NET 8)

## Overview
BDD-based UI automation framework for the **Mars Competition** web application using:

- ReqnRoll (BDD)
- Selenium WebDriver
- C# .NET 8
- Page Object Model (POM)
- JSON external test data
- Screenshot capture on test failure

Automated features include:
- Login
- Education Management
- Certification Management
  

## Framework Structure

Features/ → Gherkin feature files
StepDefinitions/ → ReqnRoll step definitions
Pages/ → Page Object classes
Drivers/ → WebDriver setup
Support/Hooks.cs→ Test hooks & screenshots
Models/ → Test data models
TestData/ → JSON test data
Utilities/ → JSON readers & helpers


## Prerequisites

- .NET 8 SDK  
- Google Chrome  
- Git
